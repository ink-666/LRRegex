## 📚 Краткая теория

Регулярные выражения — это специальные шаблоны для поиска, анализа и изменения текста.

В C# работа с регулярными выражениями осуществляется через пространство имён:

```csharp
using System.Text.RegularExpressions;
```

Главным классом является класс `Regex`, который позволяет:

- искать совпадения в тексте;
- подсчитывать символы и слова;
- проверять строки на соответствие шаблону;
- заменять части текста.

---

## 🧩 Описание программы

Программа выполняет обработку текста при помощи регулярных выражений.

В качестве исходного текста используется несколько строк:

```csharp
string text = @"
Программирование делает жизнь проще.
Регулярные выражения помогают искать текст.
Работа со строками очень важна.
Программисты часто используют Regex.
Регулярные выражения экономят время.
";
```

---

## ⚙️ Реализованные задачи

### 1. Подсчёт количества символов

Программа считает количество букв «о» в тексте.

Для этого используется регулярное выражение:

```csharp
@"о"
```

Параметр:

```csharp
RegexOptions.IgnoreCase
```

позволяет игнорировать регистр букв.

---

### 2. Подсчёт количества слов

Для поиска слов используется шаблон:

```csharp
@"\b\w+\b"
```

где:

- `\b` — граница слова;
- `\w+` — последовательность букв или цифр.

С помощью метода `Matches()` программа находит все слова и подсчитывает их количество.

---

### 3. Поиск словосочетания

Программа ищет словосочетание:

```text
Регулярные выражения
```

и выводит количество найденных совпадений.

Используемое выражение:

```csharp
@"Регулярные выражения"
```

---

### 4. Поиск строк, начинающихся с определённого слова

Для поиска строк используется символ начала строки `^`:

```csharp
@"^Регулярные.*"
```

Программа выводит строки, которые начинаются со слова «Регулярные».

Параметр `RegexOptions.Multiline` позволяет анализировать каждую строку отдельно.

---

### 5. Поиск строк, оканчивающихся определённым словом

Для поиска конца строки используется символ `$`:

```csharp
@".*время\.\$"
```

Программа выводит строки, которые заканчиваются словом «время.»

---

### 6. Замена текста

Метод `Replace()` заменяет слово:

```text
Regex
```

на:

```text
C#
```

Используемый код:

```csharp
string newText = replaceRegex.Replace(text, "C#");
```

## 💻 Используемые классы и методы

### Класс `Regex`

Основной класс для работы с регулярными выражениями.

### Метод `Matches()`

Возвращает коллекцию найденных совпадений.

### Метод `Replace()`

Заменяет найденные части строки другим текстом.

### Класс `MatchCollection`

Хранит найденные совпадения.
