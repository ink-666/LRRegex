## 🎯 О работе

В данной лабораторной работе изучается использование регулярных выражений в языке C#. Программа выполняет анализ текста, поиск совпадений по шаблонам, подсчёт слов и символов, а также замену частей текста.

Регулярные выражения позволяют эффективно работать со строками и автоматизировать обработку текстовой информации.

---

## 🧩 Описание кода

Программа написана на языке C# с использованием пространства имён:

```csharp
using System.Text.RegularExpressions;
```

Главным классом для работы с регулярными выражениями является класс `Regex`.

### Основные части программы:

#### 1. Исходный текст
В начале программы создаётся строковая переменная `text`, содержащая текст для анализа.

```csharp
string text = @"...";
```

#### 2. Подсчёт символов
С помощью метода `Matches()` программа подсчитывает количество букв "о" в тексте.

```csharp
Regex symbolRegex = new Regex(@"о", RegexOptions.IgnoreCase);
```

Параметр `IgnoreCase` позволяет игнорировать регистр букв.

#### 3. Подсчёт слов
Для поиска отдельных слов используется регулярное выражение:

```csharp
@"\b\w+\b"
```

Оно находит все слова, состоящие из букв и цифр.

#### 4. Поиск словосочетаний
Программа ищет словосочетание:

```csharp
Регулярные выражения
```

и выводит количество найденных совпадений.

#### 5. Поиск строк по началу
Используется символ `^`, обозначающий начало строки:

```csharp
@"^Регулярные.*"
```

Программа выводит строки, начинающиеся со слова «Регулярные».

#### 6. Поиск строк по окончанию
Используется символ `$`, обозначающий конец строки:

```csharp
@".*время\.$"
```

Программа выводит строки, оканчивающиеся словом «время.»

#### 7. Замена текста
Метод `Replace()` заменяет слово `Regex` на `C#`.

```csharp
string newText = replaceRegex.Replace(text, "C#");
```

Это демонстрирует обработку и изменение текста при помощи регулярных выражений.

## 📚 Теоретическая часть

### Основные элементы регулярных выражений

| Символ | Описание | Пример |
|--------|----------|--------|
| `\d` | Любая цифра | `\d{3}` - три цифры |
| `\D` | Любой нецифровой символ | `\D+` - не цифры |
| `\w` | Любая буква, цифра или подчёркивание | `\w+` - слово |
| `\W` | Любой не-буквенно-цифровой символ | `\W` - знаки препинания |
| `\s` | Любой пробельный символ | `\s+` - один или более пробелов |
| `\S` | Любой непробельный символ | `\S+` - слово без пробелов |
| `.` | Любой символ | `м.р` - "мир" или "мор" |
| `^` | Начало строки | `^Мой` - строки, начинающиеся с "Мой" |
| `$` | Конец строки | `;$` - строки, оканчивающиеся на ";" |
| `*` | 0 или более повторений | `\w*` - любое количество букв |
| `+` | 1 или более повторений | `\d+` - одна или более цифр |
| `?` | 0 или 1 повторение | `цвет?` - "цвет" или "цве" |
| `{n}` | Ровно n повторений | `\d{3}` - ровно 3 цифры |
| `{n,}` | n или более повторений | `\d{2,}` - от 2 цифр |
| `{n,m}` | От n до m повторений | `\d{2,4}` - от 2 до 4 цифр |
| `[а-я]` | Диапазон русских букв | `[а-я]+` - русские слова |
| `[a-z]` | Диапазон латинских букв | `[a-z]+` - английские слова |
| `[0-9]` | Диапазон цифр | `[0-9]{3}` - три цифры |
| `(a\|b)` | Альтернатива | `(не\|ни)` - "не" или "ни" |

### Класс Regex в C#

```csharp
using System.Text.RegularExpressions;

// Создание регулярного выражения
Regex regex = new Regex(@"\b\w+\b");

// Поиск всех совпадений
MatchCollection matches = regex.Matches(text);

// Проверка соответствия
bool isMatch = Regex.IsMatch(text, pattern);

// Замена текста
string result = Regex.Replace(text, pattern, replacement);

// Поиск с игнорированием регистра
MatchCollection matches = Regex.Matches(text, pattern, RegexOptions.IgnoreCase);
```
